scalor:scala-js-env-prov-phantomjs

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.5.0.20190502181145:scala-js-env-prov-phantomjs

Description:

Provision Scala.js JavaScript VM environment for testing: install Phantom.js binary.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: generate-test-resources.
