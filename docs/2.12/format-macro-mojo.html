scalor:format-macro

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.5.0.20190502181145:format-macro

Description:

Format sources for compilation scope=macro. A member of goal=format.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: generate-sources.
