scalor:scala-native-link-main

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.5.0.20190502181145:scala-native-link-main

Description:

Generate Scala.native runtime binary for scope=main. Provides incremental linking in M2E. A member of goal=scala-native-link.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • Binds by default to the lifecycle phase: process-classes.
